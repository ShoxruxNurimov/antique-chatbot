# Antika Chatbot Projesi

## Kurulum

1. **Backend**:
   - `server` klasörüne gidin:
     ```bash
     cd server
     ```
   - Gerekli paketleri yükleyin:
     ```bash
     npm install
     ```
   - Sunucuyu başlatın:
     ```bash
     node index.js
     ```

2. **Frontend**:
   - `client` klasörüne gidin:
     ```bash
     cd ../client
     ```
   - Gerekli paketleri yükleyin:
     ```bash
     npm install
     ```
   - React uygulamasını başlatın:
     ```bash
     npm start
     ```

3. **Uygulamayı Kullan**:
   - Tarayıcıda `http://localhost:3000` adresine gidin ve chatbotu kullanın.
